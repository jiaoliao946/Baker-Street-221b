# 基于Docsify搭建静态博客

> 本系列文章围绕Docsify从零开始讲解搭建静态博客的各方面知识技巧，大部分内容暂时只保证支持Windows系统，适用于所有略懂电脑知识、拥有基本代码能力（甚至不需要学过HTML、CSS和JavaScript）且想要搭建静态博客的读者。
