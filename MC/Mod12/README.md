# 我真的想教你开发Minecraft 1.12.2 forge模组

> 本系列文章从零开始讲解Minecraft 1.12.2 forge模组开发的各方面知识技巧，基于cleanroom和IntelliJ IDEA，适用于所有较懂MC、拥有基本代码能力（甚至不需要学过Java）、大致掌握Git使用且想要开发Minecraft 1.12.2 forge模组的读者。

### 什么是Cleanroom

​	[Cleanroom](https://www.mcmod.cn/class/9689.html)是由[CleanroomMC](https://www.mcmod.cn/author/26409.html)团队基于[Minecraft Forge](https://www.mcmod.cn/class/30.html)1.12.2制作的模组加载器分支，致力于继续维护1.12.2的Forge模组社区，其开发将侧重于对Minecraft原版以及Forge进行漏洞修复与增加功能。
